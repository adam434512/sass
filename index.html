sass introduction

sass stands for syntactically awesome stylsheets

sassis a styleesheet language that is compiled to css sass has powerful and magical features like mixin, function, nested rule etc

sass support two types of syntax. so its up to you and your team to choose up

  • scss syntax: which contains full set of css syntax with few exceptions.
  • indented synstanx: its the original sass syntanx.

variables

sass variables are variables defined and assigned the values which can be re-used throughout the stylesheet. variable make easier to change the value of the variables and also reduce repitition

mixins and include

mixins allows a developer to write a style that can be re-use throughout the stylesheet. mixins are include in the current context using the @include rule with the name of the mixin included.

sass functions

functions allo a developer to define a complex operation, which can be re-used throughout the stylesheet. functions make easy to abstract out common formulas and beahvaiors in a readable way

sass extend @extend

there is often a case when designing a page when one class should have all the style of another class as well as its own specific styles. @extend allow style to be inherited from one selector to another selector
